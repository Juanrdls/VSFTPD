# Instalación

En esta apatado vamos a realizar la instalición de VSFTPD en Debian 10:

## 1.- Actualizamos los repositorios con: ``` apt update ```

## 2.- Instalamos el paquete de VSFPTD: ``` apt install vsftpd ```
